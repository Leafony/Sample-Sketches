## ESP-AT for Leafony
ESP32 MCUをATコマンドで使用するためのファームウエア

espressifが提供している[ESP-AT](https://github.com/espressif/esp-at) V2.4.0.0(2022/05/20)のUARTピンアサインをに合わせたもの 

• TX (GPIO16)	  D9<br>
• RX (GPIO17)	  D8<br>
• CTS (GPIO15)	D7 (optional)<br>
• RTS (GPIO14)	D6 (optional)<br>


• Baudrate  : 115200<br>
• Data Bits : 8<br>
• Parity    : None<br>
• Stop Bits : 1<br>
• Flow Type : None<br>
